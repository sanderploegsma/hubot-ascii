# hubot-ascii

Convert text to ASCII

See [`src/ascii.coffee`](src/ascii.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-ascii --save`

Then add **hubot-ascii** to your `external-scripts.json`:

```json
[
  "hubot-ascii"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
